# 日本における科学研究政策史年表 (Chronology of Science and Research Policy in Japan)

日本における科学技術政策と研究支援（URA等）の変遷を辿る、モダンなタイムライン・ウェブアプリケーションです。

## 特徴
- **Premium UI**: ガラスモーフィズム、3Dホバーエフェクト、GSAPによるスムースなスクロールアニメーションを採用。
- **データ駆動**: `history.json` からデータを読み込み、年度・カテゴリー・キーワードでのフィルタリングが可能。
- **イベント編集**: ブラウザ上で新しいイベントを追加したり、不要なイベント（デフォルトデータ含む）を削除して自分好みの年表を作成できます。

## セットアップ
1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くだけで動作します。
